# RFQ Settlement

This package contains contracts, tests, sdk and utilities for the RFQ settlement contracts.

## Setup and tests

If `$pwd` is this package you can drop `workspace evm` from the commands below.

1. `corepack enable`

2. `yarn workspace evm install`

3. `yarn workspace evm compile`

4. `yarn workspace evm test`

5. `yarn workspace evm coverage`

6. `yarn workspace evm deploy`

## SDK

This package exports contract types, signing and verification utils, contract interaction helpers, deploy scripts.
